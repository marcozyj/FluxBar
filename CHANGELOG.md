# Changelog

## 0.1.1 - 2026-03-18

以稳定性与设置逻辑对齐为主的迭代版本。

### Added

- DMG 打包脚本（`Scripts/build-dmg.sh`）
- 策略页隐藏策略组显示记忆与底部分组（自动/故转）
- 配置目录清理策略与仅保留用户导入 YAML 的约束

### Changed

- 启动流程改为按上次偏好恢复系统代理状态，不再启动即重置
- 策略页“自动选择”组改为统一列表展示，不再拆分地区/其他
- 配置管理菜单文案与入口调整（当前配置名称改为配置文件，配置目录改为直达）
- 默认打包版本号更新为 `0.1.1`

### Fixed

- 修复策略弹层节点列表无法上下滚动的问题
- 修复隐藏策略组在部分配置下不显示的问题（含 anchor/运行时补全场景）
- 修复系统代理连续切换与启动恢复一致性问题
- 修复 TUN 关闭/切换过程中部分内核异常恢复路径

## 0.1.0 - 2026-03-15

首个可交付版本。

### Added

- Swift Package Manager 构建与本地 `.app` 打包流程
- 菜单栏应用壳与 Popover 主界面
- 节点、策略、分流、网络、设置页面
- `mihomo` 内核管理、controller API 接入
- 订阅刷新、节点解析、延迟测试
- 分流规则缓存与规则集解析
- 网络连接监控与连接列表
- 日志面板与内核更新检查
- 开机自启（Login Item）
- `mihomo` 内置 TUN 启动链路

### Changed

- UI 统一为白玻璃风格并向 HTML 原型靠拢
- 顶部状态区改为外部面板可点击入口
- 节点页快速总览改接真实数据源
- 策略页与节点页共用延迟缓存
- 分流页改为持久化缓存优先展示
- 网络页改为最新优先、规则集与策略组双展示
- 设置页默认值统一收敛为首次启动关闭系统代理 / IPv6 / TUN

### Fixed

- 修复首次启动默认值未写入配置的问题
- 修复开机自启仅为本地状态而未接真实行为的问题
- 修复旧目录残留导致的 SwiftPM 构建 warning
- 修复部分 controller / traffic 读取导致的总览空值问题

### Known Limitations

- `smart` 内核仍未实现
- `TUN` 仍依赖管理员授权启动，不是独立系统服务
- 系统代理模式无法覆盖所有自定义网络栈应用
