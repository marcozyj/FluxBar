# Changelog

## 0.1.0 - 2026-03-15

首个可交付版本。

### Added

- Swift Package Manager 构建与本地 `.app` 打包流程
- 菜单栏应用壳与 Popover 主界面
- 节点、策略、分流、网络、设置页面
- `mihomo` 内核管理、controller API 接入
- 订阅刷新、节点解析、延迟测试
- 分流规则缓存与规则集解析
- 网络连接监控与连接列表
- 日志面板与内核更新检查
- 开机自启（Login Item）
- `mihomo` 内置 TUN 启动链路

### Changed

- UI 统一为白玻璃风格并向 HTML 原型靠拢
- 顶部状态区改为外部面板可点击入口
- 节点页快速总览改接真实数据源
- 策略页与节点页共用延迟缓存
- 分流页改为持久化缓存优先展示
- 网络页改为最新优先、规则集与策略组双展示
- 设置页默认值统一收敛为首次启动关闭系统代理 / IPv6 / TUN

### Fixed

- 修复首次启动默认值未写入配置的问题
- 修复开机自启仅为本地状态而未接真实行为的问题
- 修复旧目录残留导致的 SwiftPM 构建 warning
- 修复部分 controller / traffic 读取导致的总览空值问题

### Known Limitations

- `smart` 内核仍未实现
- `TUN` 仍依赖管理员授权启动，不是独立系统服务
- 系统代理模式无法覆盖所有自定义网络栈应用
